About the Project

The UNCC Animal Surveying Project is an initiative led by a group from the Honors College at the University of North Carolina at Charlotte. Our mission is to explore various sections of campus and collect data on animal species that inhabit our surroundings. This ongoing research aims to create a comprehensive database to help track how ecosystems and food webs evolve as campus development progresses over time.

Purpose of this website

This site will serve as a central space for all the data to be collected throughout this project on the speciest within the UNC Charlotte campus.

Intended Users

This site is intended to be used by the researcher or others who would like to be involved in the research.

Dynamic functionality

on the projects page there is a jquery that allows for the user to select and submit a date to volunteer for the project

there is an AJAX functionality on the contact page that allows for the email to be sent without having to reload the page.
